Quilt.js

A JavaScript framework to build, deploy, and manage anything from website backends to complex distributed systems.

Infrastructure with JavaScript

Quilt aims to be the easiest way to deploy and manage applications in the cloud. With Quilt.js, you can structure and develop infrastructure as you develop applications.

Develop Services Independently

Treating each subsystem (e.g. MongoDB or ElasticSearch) as a seperate module allows you to create services separately and easily compose them through APIs that expose dependencies while hiding low-level details.

Share and Reuse

Import any Quilt specifications posted on GitHub and compose them to create your own infrastructure. No need to reinvent the wheel and reimplement services that have been deployed thousands of times already.
